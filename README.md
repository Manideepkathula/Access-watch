# AccessWatch — Emergency Access Equity Index

A free public report card scoring every major US city on emergency response fairness and naloxone access equity.

**Live site:** https://accesswatch.vercel.app

## What it measures
1. **EMS Response Equity** — How evenly ambulance/911 response times are distributed across rich vs poor neighborhoods
2. **Naloxone Access** — How easy it is to get overdose-reversal medicine in the area

## Data sources
- CDC IQVIA Naloxone Dispensing Rate Maps 2024
- NEMSIS 2024 Public Release Research Dataset (60M+ EMS activations)
- NYC/Chicago/LA/Boston/Seattle Open Data Portals (live APIs)
- US Census Bureau ACS 2023

## Auto-updates
Data refreshes every 3 months automatically via GitHub Actions.
